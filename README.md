# PMDG-P3D-livery-installer
A rudimentary program to install liveries for PMDG's P3D products. Can unpack PTP files and install the livery into any version of Prepar3D for the 737 NGX/NGXu, 777X, 747 QOTS II and DC-6. Can also be used to unpack PTP files for Microsoft Flight Simulator 2020/2024 or FSX.
